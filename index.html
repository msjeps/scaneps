Instructions pour exporter vers Google Sheets :

  1. Cliquez sur le bouton ci-dessous pour télécharger le fichier Excel
  2. Ouvrez Google Drive sur votre appareil
  3. Cliquez sur "+ Nouveau" puis "Importer un fichier"
  4. Sélectionnez le fichier téléchargé
  5. Une fois importé, faites clic droit sur le fichier
  6. Choisissez "Ouvrir avec" → "Google Sheets"

💡 Astuce : Le fichier sera automatiquement converti en Google Sheets !
